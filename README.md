# Twenty20 Portfolio

A modern portfolio web application with authentication built for Twenty20 Systems assessment.

## Features
- User Registration & Login
- JWT Authentication
- MongoDB Database
- Responsive Black & Red Theme
- Protected Portfolio Page

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js (Vercel Serverless)
- Database: MongoDB Atlas
- Deployment: Vercel
